# Pulse

Pulse is a premium local media player for Android, Windows, and Web.

## Phase 1

The repository currently contains the product foundation and official distribution website.

## Local Commands

```bash
npm test
npm run build
npm run serve
```

The local site serves at `http://127.0.0.1:4173`.

## Structure

- `website/` - distribution website
- `docs/` - brand, changelog, and release strategy
- `app/` - Flutter app workspace notes until the Flutter SDK is available
- `scripts/` - local verification and build scripts
