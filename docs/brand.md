# Pulse Brand Guide

## Product Name

Pulse

## Positioning

Pulse is a premium local media player for Android, Windows, and Web. It should feel calm, fast, polished, customizable, and deeply respectful of the user's library.

## Visual Identity

- Primary background: near-black graphite `#101114`
- Primary text: warm white `#f7f3eb`
- Accent: signal orange `#ff6b3d`
- Secondary accent: aqua mint `#55d6be`
- Light surface: warm porcelain `#f8f4ee`
- Shape language: rounded, soft, precise, never bulky
- Shadows: large and diffuse, used sparingly
- Blur: reserved for navigation, player surfaces, and optional Glass theme

## Typography

Use Inter or the platform system sans-serif stack for the website. The Flutter app should expose font selection later, with Inter as the baseline Latin font.

## Icon Style

Use simple rounded line icons with filled emphasis only for active player controls. Icons should remain recognizable at small sizes.

## Motion Philosophy

Motion is subtle and useful. Prefer 150-300ms transitions, soft easing, hero movement for media artwork, and low-amplitude scale/fade changes. Avoid decorative motion that competes with playback.
